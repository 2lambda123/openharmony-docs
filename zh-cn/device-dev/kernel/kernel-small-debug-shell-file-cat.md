## 命令功能

cat用于显示文本文件的内容。


## 命令格式

cat [_pathname_]


## 参数说明

**表1** 参数说明

| 参数 | 参数说明 | 取值范围 | 
| -------- | -------- | -------- |
| pathname | 文件路径。 | 已存在的文件。 | 


## 使用指南

cat用于显示文本文件的内容。


## 使用实例

举例：cat hello-harmony.txt


## 输出说明

**示例**：查看 hello-harmony.txt 文件的信息
```
OHOS # cat hello-harmony.txt
OHOS # Hello Harmony ;)
```
