# 媒体资源类型说明



- 开发框架支持的图片资源类型说明
  | 格式 | 文件后缀名 | 
  | -------- | -------- |
  | JPEG | .jpg | 
  | PNG | .png | 
  | GIF | .gif | 
  | SVG | .svg | 
  | WEBP | .webp | 
  | BMP | .bmp |

- 开发框架支持的图片资源类型说明
  | 格式 | 支持的文件类型 | 
  | -------- | -------- |
  | H.263 | .3gp <br>.mp4 | 
  | H.264 AVC <br> Baseline Profile (BP) | .3gp <br>.mp4 | 
  | MPEG-4 SP | .3gp | 
  | VP8 | .webm <br> .mkv |
