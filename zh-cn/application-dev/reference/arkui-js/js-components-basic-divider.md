# divider<a name="ZH-CN_TOPIC_0000001127125060"></a>

提供分隔器组件，分隔不同内容块/内容元素。可用于列表或界面布局。

## 权限列表<a name="section11257113618419"></a>

无

## 子组件<a name="section9288143101012"></a>

不支持。
