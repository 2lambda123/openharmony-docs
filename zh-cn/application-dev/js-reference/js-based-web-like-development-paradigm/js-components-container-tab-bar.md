# tab-bar<a name="ZH-CN_TOPIC_0000001173324603"></a>

<[tabs](js-components-container-tabs.md)\>的子组件，用来展示tab的标签区，子组件排列方式为横向排列。

## 权限列表<a name="section11257113618419"></a>

无

## 子组件<a name="sfddaafa400fa4802a6c4344a0cc0f9ed"></a>
